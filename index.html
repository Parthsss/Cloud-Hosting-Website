The Airborne Web Services Platform

Airborne Web Services provides on-demand cloud computing platforms and APIs to individuals, companies, and governments, on a metered, pay-as-you-go basis. Clients will often use this in combination with autoscaling (a process that allows a client to use more computing in times of high application usage, and then scale down to reduce costs when there is less traffic). Airborne Web Services provide various services related to networking, compute, storage, middleware, IoT and other processing capacity, as well as software tools via server farms.
